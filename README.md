# Boost.Trie

Boost containers based on the trie data structure.

# Build

    mkdir build
    cd build
    cmake -DBOOST_ROOT=/path/to/boost/root ..
    make tests


#[Boost.Trie](http://boost.org/libs/trie)
Boost.Trie, it is a set/map-like container based on the Trie data structure.
